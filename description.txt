1 - Inital Clone Test 
    clone the learning repo from the git hub url by executing : 
    git clone https://github.com/8492ed/git_learning.git

2 - add, ,commit ,push test 
    create a folder with your name inside the repo
    inside the that folder create a file called `test.txt`
    and write your email inside it 
    in order to save the file to the repo 
    we should add it : git add 'file path'
    we should commit it : git commit -m 'message'
    finally we should push it : git push origin master
    
3 - pull finally to see the changes 
    to get the updates from the repo we execute `git pull origin <branche name>`

4 - branches managment
    `git branche <name>` to create bransh 
    `git checkout <existing_branch>` to switch branche

5 - save changes & restore original state 
    `git stash` to stash changes
    `git stah pop` to restore changes
